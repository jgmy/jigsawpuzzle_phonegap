Purpose

This project is simply a fork of JigsawPuzzle, intended for being auto-exported to phone apps by phonegap.

Downloading

You can download the mobile apps here

More details

JigsawPuzzle is a simple html5 jigsaw made with construct2. It can use two different tilesets.

It can use three different jigsaw pictures. More pictures could be added upon request, but take in account they consume memory in the resulting app.

Customizing

If you want to build your own flavored app with your own image, fork this repository, add your image and edit the lista_xml.xml file, adding your image name, description, license and original url. Please notice the url field is used exclusively as a license reference. Due javascript security restrictions, all images should be located in the same server where your app is.
